Inevitable is a 4-player 3rd person runner.

Each Character has their own ability (with cooldowns) that allows them to affect all other players or themselves.
Players must run to avoid falling in order to be the last one standing.

Abilities
	Ozean - Teleport 
			Holding the ability button shows potential teleport spot, releasing activates it.
	Zeit - Slow
			Slows every other character for a set time.
	Brick - Invincibility
			Can't be affected by abilities for a set time.
	Lyn - Knockback
			Knocks other characters away from herself for a set time.

Note: Controllers must be plugged in prior to launching the game.
